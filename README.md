# Human-Care

Human-Care is a Patient Appointment Booking System developed using PHP, HTML, CSS, and JavaScript.

This project allows users to select doctor, doctor type, date, and time to schedule an appointment.

Features:
- Select doctor
- Select doctor type
- Select date
- Select time
- Simple appointment form

Technologies Used:
- PHP
- HTML
- CSS
- JavaScript

How to run:
1. Copy project folder into htdocs (XAMPP)
2. Start Apache in XAMPP
3. Open browser
4. Go to http://localhost/human-care
